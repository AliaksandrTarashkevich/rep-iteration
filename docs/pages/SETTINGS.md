# Settings Page (/settings)

## Purpose
User settings and account management.

## Sections

### 1. Connected Accounts
- X (Twitter) - connect/disconnect
- Wallet (Base) - connect/disconnect
- Telegram - connect/disconnect (for notifications)
- Solana - connect/disconnect
- EVM (other chains) - connect/disconnect

### 2. Notifications
- Email notifications toggle
- Telegram notifications toggle (requires Telegram connection)
- Weekly digest toggle

### 3. Theme
- Dark (default)
- Light

### 4. Language
- English (default)
- Future: other languages

### 5. Account Actions
- Log out
- Export data (future)
- Delete account (future)

## Components

### 1. Section Headers
- Clear section titles
- Optional descriptions

### 2. Connected Account Row
- Service icon
- Service name
- Connection status
- Connect/Disconnect button

### 3. Toggle Settings
- Setting name
- Description
- Toggle switch

### 4. Theme Selector
- Visual preview
- Radio buttons or toggle

### 5. Danger Zone
- Log out button
- Delete account (with confirmation)

## Design Notes

- Clean, organized layout
- Clear visual feedback for connected/disconnected states
- Confirmation dialogs for destructive actions
- Mobile-friendly spacing
