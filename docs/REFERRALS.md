# Referrals System

> **Note:** Details require further discussion. Finalize at next sync.

## Code Format

- 6-7 characters
- Alphanumeric
- Case-insensitive
- **Text codes, NOT links** (Twitter deprioritizes links)

## Code Allocation

- 3-5 codes per user
- Refresh weekly with rank growth

## Referral Tree

- Multi-degree tracking (up to 4 levels)
- Gen 1: 10%
- Gen 2: 5%
- Gen 3: 2%
- Gen 4: 1%
- Capped at 40% of organic gravity

## URL Format

```
rep.xyz/@handle?ref=@handle
```
or
```
r3p.xyz/invite/abc123
```

## Tracking

- Referral tree with attribution
- Codes work as UTM tags (tracking who brought whom)

## Components

### 1. Referral Codes Section (in Profile or separate page)
- List of user's codes
- Copy button for each
- Usage stats per code

### 2. Referral Tree View
- Visual tree of referrals
- 4 generations deep
- Stats per generation

### 3. Share Modal
- Pre-formatted text with code
- Twitter share intent
- Copy to clipboard
