# Navigation & Layout

## Desktop Layout

- Fixed sidebar (left)
- Main content area (right)

### Sidebar - Authenticated

**Main Items:**
- Profile
- Achievements
- Leaderboard
- Chats

**Secondary Items:**
- Manifesto
- Insights
- Settings
- Log out

**Footer:**
- Mini user card (avatar, @handle, REP score)

## Mobile Layout

### Bottom Tab Bar
- Profile
- Achievements
- Chats
- Leaderboard

## Roles & Permissions

| Role | Description |
|------|-------------|
| Visitor | Sees landing, can start onboarding |
| Onboarded User | Sees web space, profile, achievements, leaderboard |
| Eligible Creator | Can create chat (high-rank or $5) |
| Chat Owner | Edits achievement set in mini-admin |
| Ops/Admin | Override access, monitoring, emergency actions |

## Theme

- Dark (default)
- Light (optional)

## Language

- English (default)
- Multilanguage support (future)
